CoreAuth is a lightweight, modern and fully object-oriented user authentication and profile management system for PHP 8.1+ applications.Built with simplicity and security in mind, CoreAuth provides everything you need to handle user registration, login, password hashing (bcrypt/argon2), email verification, password reset, session management, remember-me functionality, and complete profile management — all in a clean, extensible OOP structure.Key features:Secure password hashing with PHP’s native password_* functions  
Role and permission support (RBAC-ready)  
Event-driven design (hooks for login, logout, registration, etc.)  
Middleware-ready for PSR-7/PSR-15 projects  
Database-agnostic (ships with PDO + MySQL examples, easily adaptable)  
Zero external dependencies  
Clean separation of concerns with dedicated classes for Auth, User, Session, and Profile

Perfect for small to medium projects, custom CMSs, admin panels or as a secure starting point when you don’t want the overhead of full frameworks like Laravel.Drop it into any PHP project and have a production-ready authentication system in minutes.
